# Agile Web Foundations Project

This is a multi-page interactive web application built using HTML, CSS, and JavaScript.

## Features
- Tutorial page for HTML, CSS, JavaScript
- Dynamic quiz using AJAX
- Randomized questions
- Score calculation and pass/fail logic
- Reward using public API
- LocalStorage for attempt history
- AI Reflection page
- Personal CV page

## Technologies Used
- HTML5
- CSS3 (Bootstrap + custom CSS)
- JavaScript (ES6)
- Fetch API
- LocalStorage
